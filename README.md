# 🚗 Smart Parking System

## 📌 Project Overview
This project is a Smart Parking System built using Node.js, Express.js, and MongoDB.  
It allows efficient management of parking slots, vehicle entry and exit, and real-time tracking of parking status.

---

## ⚙️ Features
- Parking slot initialization
- Automatic slot allocation
- Manual slot selection
- Vehicle entry and exit tracking
- Parking history maintained
- Middleware logs API requests
- CRUD operations supported
- APIs tested using Postman

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- HTML, CSS (Frontend)
